Jadual ini telah disusun mengikut agihan yang seimbang bagi Imam Aiman, Imam Faiz, dan Imam Naudi.

## Panduan Warna Google Sheets
Untuk hasil seperti dalam gambar, gunakan kod Hex berikut:
- **Ungu Gelap (Hari):** `#4B0082` (Teks Putih)
- **Kuning Terang (Header):** `#FFFF00`
- **Peach (Aiman):** `#FFDAB9`
- **Kuning Mustard (Faiz):** `#FFDB58`
- **Cyan (Naudi/Duet):** `#00FFFF`
- **Kelabu (Standby):** `#D3D3D3`
